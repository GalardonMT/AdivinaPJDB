# Proyecto: Juego de Adivinar Personajes de Dragon Ball

Este proyecto es un sencillo juego web en el que el usuario debe adivinar el nombre de un personaje de Dragon Ball. 
El juego está desarrollado con HTML, CSS y JavaScript y está pensado como práctica para lógica de programación y manipulación del DOM.

## Estructura del proyecto

- `juegoAdivinarPJ.html`: página principal del juego.
- `css/style.css`: estilos del juego.
- `js/juegoAdivinarPJ.js`: lógica del juego (selección de personajes, validación de respuestas, etc.).
- `assets/`: recursos estáticos (imágenes, íconos, etc.).

## Funcionalidades previstas

- Mostrar un personaje de Dragon Ball (por imagen o pista) para que el usuario adivine su nombre.
- Validar la respuesta del usuario y mostrar mensajes de acierto o error.
- Cambiar de personaje después de cada intento o ronda.

## Cómo ejecutar el proyecto

1. Clonar o descargar este proyecto en tu computadora.
2. Abrir el archivo `juegoAdivinarPJ.html` directamente en el navegador.
3. Interactuar con la interfaz para intentar adivinar los personajes.
